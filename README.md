# Customised App V3 - أحمد المحاميد

## الجديد في V3
- تحديد الموقع الحالي GPS
- اتجاه القبلة
- التاريخ الهجري
- إعدادات تنبيهات ذكية:
  - قبل الصلاة بـ 10 دقائق
  - وقت الصلاة
  - تشغيل/إيقاف لكل صلاة
- تجربة صوت التنبيه
- تحسينات على التصميم

## ملاحظات مهمة
- لو أردت أذان حقيقي، أضف ملف باسم `adhan.mp3` بجانب الملفات.
- إذا لم يوجد ملف الأذان، التطبيق يشغل صوت تنبيه بسيط.
- تنبيهات المتصفح تعمل أفضل عندما يكون التطبيق مفتوحاً أو حديث الاستخدام.
- البوصلة تعتمد على دعم الجهاز والمتصفح.
