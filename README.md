# Customised App V2 - أحمد المحاميد

المزايا:
- اختيار المدينة
- الشروق والغروب
- أوقات الصلاة
- الطقس
- الوضع الداكن
- تنبيهات للصلاة
- تصميم PWA قابل للتثبيت على الموبايل

التعديل:
- التصميم: index.html
- المدن والحسابات والمنطق: app.js
- اسم التطبيق والأيقونة: manifest.json و icon.svg

ملاحظة: التنبيهات تعمل أفضل عندما يكون التطبيق مفتوحاً أو حديث الاستخدام. التنبيهات الدائمة بالخلفية تحتاج Push Server لاحقاً.
