# شروق وغروب الدوحة - PWA

هذا تطبيق ويب بسيط يعمل كتطبيق على الموبايل.

## الملفات
- index.html: واجهة التطبيق
- app.js: كود الحساب والتحديث
- manifest.json: إعدادات تثبيت التطبيق
- service-worker.js: تشغيل Offline وكاش
- icon.svg: أيقونة التطبيق

## التعديل
أي تعديل على الشكل يكون غالباً داخل index.html.
أي تعديل على الحسابات أو الموقع يكون داخل app.js.

## التشغيل
ارفع كل الملفات معاً على GitHub Pages أو Netlify أو Tiiny Host.
بعدها افتح الرابط من Chrome واضغط:
Add to Home Screen
