# Customised App V5 Final - أحمد المحاميد

## الجديد
- طقس احترافي: حرارة، إحساس، رطوبة، رياح، مطر.
- اختيار طريقة الصلاة: Qatar / Umm Al-Qura / Muslim World League.
- نوم ذكي حسب الفجر والعشاء وهدف النوم.
- أذكار الصباح والمساء.
- صفحة أدعية قابلة للإضافة والبحث.
- تحسين القبلة والبوصلة.

## ملاحظات
- التنبيهات بالخلفية الكاملة تحتاج Firebase/OneSignal لاحقاً.
- دقة القبلة تعتمد على GPS + حساس البوصلة + المتصفح.
- لإضافة أذان حقيقي، أضف ملف adhan.mp3 بجانب الملفات.
